================================================================================
                        API ENDPOINTS
================================================================================

Your Laravel API needs these endpoints:

AUTHENTICATION:
POST   /api/auth/login              → Returns { token, user }
POST   /api/auth/logout             → Clears auth
GET    /api/auth/me                 → Returns current user

PRODUCTS:
GET    /api/products                → Returns list of products
GET    /api/products/{id}           → Returns single product
GET    /api/categories              → Returns categories
GET    /api/categories/{id}/products → Returns products by category
POST   /api/products                → Create product (admin)
PUT    /api/products/{id}           → Update product (admin)
DELETE /api/products/{id}           → Delete product (admin)

TRANSACTIONS:
POST   /api/transactions            → Create POS transaction
GET    /api/transactions            → List transactions (admin)
GET    /api/transactions/{id}       → Get transaction detail
GET    /api/transactions/report     → Get report (admin)
GET    /api/transactions/daily-revenue → Get daily revenue (admin)

See LARAVEL_API_SETUP.md for complete specification with response examples.

================================================================================
                        TROUBLESHOOTING
================================================================================

Q: Port 3000 already in use?
A: npm run dev -- -p 3001

Q: API connection error?
A: Check NEXT_PUBLIC_API_URL in .env.local
   Verify Laravel API is running
   Check LARAVEL_API_SETUP.md

Q: Styling looks broken?
A: rm -rf .next && npm run dev

Q: Images not showing?
A: Check public/images/products/ folder
   Clear browser cache
   Restart dev server

Q: Still have issues?
A: Check relevant documentation file
   Review error messages in console
   See LARAVEL_API_SETUP.md for API issues
