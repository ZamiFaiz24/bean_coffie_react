CUSTOMIZATION EXAMPLES
================================================================================

Jika ingin customize:

1. Ubah tax rate (sekarang 10%)
   Cari: const tax = subtotal * 0.1;
   Ubah 0.1 ke nilai yang diinginkan

2. Ubah layout grid produk
   Cari: grid-cols-2 md:grid-cols-3
   Ubah sesuai kebutuhan

3. Ubah warna tema
   Edit globals.css
   Update design tokens: --primary, --background, etc

4. Ubah mock data
   Edit data di useEffect untuk development

5. Ubah format currency
   Sekarang: .toLocaleString('id-ID')
   Bisa disesuaikan locale
